# PI.-Aplicaci-n-Web
# README — Sistema Web de Catálogo de Películas

## Descripción

Aplicación web desarrollada con Node.js, Express y MySQL para administrar un catálogo de películas mediante operaciones CRUD protegidas con autenticación JWT.

## Tecnologías utilizadas

* Node.js
* Express
* MySQL
* JWT
* bcrypt
* HTML, CSS y JavaScript

## Funcionalidades

* Registro e inicio de sesión
* Autenticación con JWT
* CRUD de películas
* Protección de rutas privadas
* Conexión a base de datos MySQL

## Instalación Backend

```bash
npm install
npm run dev
```

## Instalación Frontend

Abrir el archivo:

```plaintext
index.html
```

## Base de Datos

Crear la base de datos:

```sql
CREATE DATABASE catalogo_peliculas;
```
